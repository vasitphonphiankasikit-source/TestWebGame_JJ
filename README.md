<h1>เกมจับคู่หรรษา</h1>
link game: https://test-web-game-jj.vercel.app/

ผมอยากสร้างเกมที่ทั้งสนุกและช่วยฝึกสมองได้ด้วย จึงเลือกเกมจับคู่ที่เล่นได้ทุกเพศทุกวัย โดยใช้ HTML, CSS และ JavaScript ที่ผมเรียนรู้มา
<h2>การออกแบบ,ฟังก์ชัน</h2>
<ul>
   <li>พื้นหลัง gradient เคลื่อนไหว</li>
   <li>ปุ่มเริ่มเกมมี animation เด้ง</li>
   <li>การ์ดเปลี่ยนสีเมื่อเปิดหรือจับคู่สำเร็จ</li>
   <li>แสดง emoji ฝนตก เมื่อเล่นจบเสร็จสิ้น</li>
   <li>JavaScript ให้ควบคุมการสุ่มสัญลักษณ์ ตรวจจับคู่ จับเวลา และแสดงผล</li>
   <li>ปรับแอนิเมชันปุ่มให้หยุดขณะเล่นเพื่อไม่รบกวนสายตา</li>
</ul>

<h2>การแก้ปัญหา</h2>
<ul>
  <li>emoji ตกไม่ตรงบนมือถือ → ใช้ position: fixed และสุ่มตำแหน่ง</li>
  <li>การ์ดไม่พอดีกับหน้าจอ → ใช้ media query ปรับตาม grid</li>
  <li>ปุ่มรบกวนตอนเล่น → เพิ่ม class เพื่อหยุดแอนิเมชันระหว่างตอนเล่น</li>
</ul>

<h2>สิ่งที่ได้เรียนรู้</h2>
 เข้าใจการจัดการ DOM, event, ฟังก์ชันแบบ modular และ responsive UI พร้อมฝึกการคิดเชิงตรรกะและการแก้ปัญหาแบบนักพัฒนา
<h2>ความภาคภูมิใจ</h2>
เกมที่ผมสร้างไว้ คือผลลัพธ์จากความตั้งใจและการเรียนรู้ ผมหวังว่าจะเป็นแรงบันดาลใจให้คนอื่นนำความรู้ไปสร้างสิ่งดี ๆ เช่นกัน
